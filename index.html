Mi primera página accesible

La accesibilidad web es una práctica fundamental que permite que todas las personas, independientemente de sus capacidades o discapacidades, puedan acceder y utilizar los sitios web. Esto incluye a personas con discapacidades visuales, auditivas, motoras o cognitivas. Implementar características como alto contraste y ajuste de tamaño de fuente son pasos simples pero significativos para lograr una web más inclusiva y equitativa para todos los usuarios.
